# Amity webgui

# Developement

To get started, you need latest Node.js LTS and `npm`.

This project contains submodule so I recommend you to do this

```sh
git clone --recurse-submodules -j8 https://github.com/negative-zero-inft/amity-webgui

cd amity-webgui
npm i
npm run dev
```
